# Cav API

### Prerequisites

What things you need to install the software and how to install them

```
Maven 3.x
docker and docker-compose

```

### Installing

Compiling the project

```
mvn clean install
```

Running the project

```
docker-compose up
```

You can also check Swagger Page in
```
http://{url}:8081/swagger-ui.html#/
```
